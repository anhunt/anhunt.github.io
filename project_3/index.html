About

This console visualizes near‑real‑time solar wind and geomagnetic activity from NOAA SWPC. The “forecast” is a playful, deterministic reading based on the current conditions and is not to be taken seriously.

Alert and Disturbance are mapped to KP ranges.

For the visualization, actual solar wind data is mapped to visual properties:

  • Speed influences how fast streaks move
  • Density influences streak thickness
  • Temperature influences color

Remember: This is an artistic mapping of public data; not a physical simulation.
