About

This console visualizes near‑real‑time solar wind and geomagnetic activity from NOAA SWPC. The “forecast” text is a playful, deterministic reading based on the current conditions.

For the visualization, actual solar wind data is mapped to visual properties:

  • Speed influences how fast streaks move
  • Density influences streak thickness
  • Temperature influences color
  • Kp sets alert level and disturbance level

Disclaimer: artistic mapping of public data; not a physical simulation.
